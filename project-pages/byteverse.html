ByteVerse
ByteVerse was an application developed in group, which explores some of the SpriteKit and CoreMotion frameworks. The main idea of the game is for the user to use the iPhone's gyroscope sensors to control the character and shoot the drones that appear on the screen. Are you ready to save the world from evil drones? The app is now available for download on the AppStore! Download now and embark on this exciting journey to victory!
